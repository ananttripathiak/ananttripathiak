## 🚀 Quick Start (Local Development)

### Option 1: Direct Open (Simplest)
```bash
# Just double-click demo.html in the folder
```

### Option 2: Local Server (Best Experience)
```bash
# Using Python
python server.py

# OR use Python's built-in server
python -m http.server 8000

# Then visit: http://localhost:8000
```

### Option 3: VS Code Live Server
1. Install "Live Server" extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## 🌐 Deployment

### GitHub Pages (Current Deployment)

**Repository:** [https://github.com/ananttripathiak/Anant-Tripathi](https://github.com/ananttripathiak/Anant-Tripathi)

**Steps to Deploy/Update:**
```bash
# Navigate to portfolio folder
cd "Anant_New Portfolio"

# Initialize git (first time only)
git init

# Add all files
git add .

# Commit changes
git commit -m "Update portfolio"

# Add remote (first time only)
git remote add origin https://github.com/ananttripathiak/Anant-Tripathi.git

# Push to GitHub
git branch -M main
git push -f origin main

# Enable GitHub Pages:
# Settings → Pages → Source: main branch, / (root) → Save
```

### Other Deployment Options

#### Netlify (Drag & Drop)
1. Visit [netlify.com](https://netlify.com)
2. Drag the entire folder
3. Instant deployment!

#### Vercel
```bash
npm install -g vercel
cd "Anant_New Portfolio"
vercel
```
